What is NAPS?

NAPS stands for Natural-language Article Processing System.

What makes it tick?

PDFMiner - A library for extracting data from PDFs.

NLTK - the Natural Language Tool Kit, contains functions for tokenizing text and converting words into stemmed form.

Django - A web framework for our application.

AWS - Amazon Web Services keeps out server up and running.
